# Security Policy

## Supported Versions

Currently, only the latest release is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in FLIR Thermal Analyzer, please report it to us via email at [INSERT CONTACT EMAIL].

Please include:
* A description of the vulnerability and its impact
* Steps to reproduce the issue
* Any potential mitigation you might have found

We will try to respond to your report within 48 hours. If the issue is confirmed, we will release a patch as soon as possible and credit you for the discovery.
