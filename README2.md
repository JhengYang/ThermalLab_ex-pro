  當然可以！如果您的伺服器要在區域網路（例如辦公室或家裡的 Wi-Fi）內讓其他設備穩定連線，將 Mac 設定為「固定 IP（靜態 IP）」是很好的作法，這樣 IP 就不會因為路由器重新開機而跑掉。

  不過，在設定固定 IP 之前，其實您還有一個更簡單的替代方案：

  ### 💡 替代方案：使用 Mac 的主機名稱 (最簡單，免設 IP)

  Apple 設備預設有支援 .local 網域（Bonjour 服務）。您可以直接使用 Mac 的電腦名稱來連線，完全不需要背 IP 或設定固定 IP！

  1. 到 Mac 的 「系統設定」 > 「一般」 > 「關於」，在底部找到 「本機主機名稱」（例如：oujy-MacBook-Pro.local）。
  2. 同一個網路下的設備，只要在瀏覽器輸入：http://oujy-MacBook-Pro.local:5050 就可以連線了！

  如果您依然需要設定固定 IP，請按照以下步驟在 Mac 上進行設定：

  ### 如何在 Mac 上設定固定 IP (手動 IP)

  步驟 1：先記下您目前的網路數值
  在改成手動之前，先記下路由器自動配發給您的數值。

  1. 點擊左上角蘋果  > 系統設定 (System Settings) > 網路 (Network)。
  2. 點擊您目前使用的網路（Wi-Fi 或乙太網路）。
  3. 點擊旁邊的 詳細資訊... (Details...)。
  4. 切換到 TCP/IP 分頁。
  5. 將目前的 「IP 位址」、「子網路遮罩」 以及 「路由器」 這三組數字記下來（等等會用到）。

  步驟 2：更改為固定 (手動) IP

  1. 在同一個 TCP/IP 頁面中。
  2. 找到 設定 IPv4 (Configure IPv4)，將其從 使用 DHCP 改為 手動 (Manually)。
  3. 欄位會變成可編輯狀態：
      • IP 位址：輸入您想要的固定 IP（建議填寫剛剛記下的 IP，或是路由器分配範圍內且沒人使用的 IP，例如 192.168.1.X，X 隨便取一個 2~254 中您喜歡的數字）。
      • 子網路遮罩：填入剛剛記下的數值（通常是 255.255.255.0）。
      • 路由器：填入剛剛記下的數值（通常是 192.168.1.1 或 192.168.0.1）。
  4. 點擊 好 (OK) 保存。

  設定完成後，只要您的 run_server.sh 裡一樣是維持 0.0.0.0:5050，您就可以在其他設備上，透過您剛剛設定的這個固定 IP（例如 http://192.168.1.50:5050）穩定連線到您的 FLIR Server 了！
